<article id='variants'>
### Variants

Use any of the available button style types to quickly create a styled button. Just modify the variant prop.

```js
const variants = {
  primary: "Primary",
  secondary: "Secondary",
  success: "Success",
  warning: "Warning",
  danger: "Danger",
  info: "Info",
  light: "Light",
  dark: "Dark",
  text: "Text",
  link: "Link",
};
```

> Note: Button has `variant='primary'` by default. Ignore if you want to use `primary` button

<Preview>
  {Object.entries(variants)?.map(([variant, label]) => (
    <Button variant={variant} onClick={() => alert(`${label} Button Clicked!`)}>
      {label}
    </Button>
  ))}
</Preview>

```js
import Button from "components/button/Button";

const VariantsTypeExample = () => {
  return (
    <>
      {Object.entries(variants)?.map(([variant, label]) => (
        <Button variant={variant}>{label}</Button>
      ))}
    </>
  );
};
```

</article>

<article id='outline'>
### Outline Buttons

Pass `outline` prop to the `<Button />`

> Note: Button with `outline` takes `variant='primary'` by default.

<Preview>
  {Object.entries(
    exclueValuesFromListOrObject(variants, ["text", "link"])
  )?.map(([variant, label]) => (
    <Button variant={`outline-${variant}`}>{label}</Button>
  ))}
</Preview>

```js
import Button from "react-bootstrap/Button";

function OutlineTypesExample() {
  return (
    <>
      {Object.entries(
        exclueValuesFromListOrObject(variants, ["text", "link"])
      )?.map(([variant, label]) => (
        <Button variant={`outline-${variant}`}>{label}</Button>
      ))}
    </>
  );
}

export default OutlineTypesExample;
```

</article>

<article id='icons'>
### Icons

The `startIcon` and `endIcon` props helps to insert your icons inside the `<Button/>`

<Preview>
  <Button startIcon={<TrashIcon />}>Delete</Button>
  <Button endIcon={<UploadIcon />}>Upload</Button>
</Preview>

```js
import Button from "components/button/Button";

import { ReactComponent as TrashIcon } from "your-assets/trash.svg";
import { ReactComponent as UploadIcon } from "your-assets/upload.svg";

const ButtonIconsExample = () => {
  return (
    <>
      <Button startIcon={<TrashIcon />}>Delete</Button>
      <Button endIcon={<UploadIcon />}>Upload</Button>
    </>
  );
};
```

> Note: You can also pass `Icons` as children to the `Button` like this

```js
<Button>
  <UploadIcon /> Upload
</Button>
```

</article>

<article id='disabled'>
### Disabled

Make buttons look inactive by adding the `isDisabled` prop.

<Preview>
  <Button isDisabled>Primary</Button>
  <Button isDisabled variant="secondary">Secondary</Button>
  <Button isDisabled variant="outline">Outlined</Button>
</Preview>

```js
import Button from "components/button/Button";

const DisabledButtonsExample = () => {
  return (
    <>
      {Object.entries({
        primary: "Primary",
        secondary: "Secondary",
        outline: "Outlined",
      })?.map(([variant, label]) => (
        <Button isDisabled variant={variant}>
          {label}
        </Button>
      ))}
    </>
  );
};
```

</article>

<article id='loading'>
### Loading State

When activating an asynchronous action from a button it is a good UX pattern to give the user feedback as to the loading state, this can easily be done by adding `isLoading` prop to `<Button />`.

<Preview>
  <Button isLoading>Primary</Button>
  <Button isLoading variant="secondary">Secondary</Button>
  <Button isLoading variant="outline">Outlined</Button>
</Preview>

```js
import Button from "components/button/Button";

const ButtonLoadingExample = () => {
  return (
    <>
      {Object.entries({
        primary: "Primary",
        secondary: "Secondary",
        outline: "Outline",
      })?.map(([variant, label]) => (
        <Button isLoading variant={variant}>
          {label}
        </Button>
      ))}
    </>
  );
};
```

</article>

<article id="loaderPosition">
### Loader Position

The loading icon can be placed at the `start` | `end` position using the `loaderPosition` prop.

> Note: By default `loaderPosition` `=` `'start'`.

<Preview>
  <Button isLoading loaderPosition="start">Save</Button>
  <Button isLoading loaderPosition="end">Save</Button>
</Preview>

```js
<Button isLoading loaderPosition='start'>Save</Button>
<Button isLoading loaderPosition='end'>Save</Button>
```

</article>

<article id='loaderAnimation'>
### Loader Animation

The `loaderAnimation` prop alow to change the type of animation for the loader.

Supports: `'border'` | `'grow'`

> Note: By default `loaderPosition` `=` `'border'`.

<Preview>
  <Button isLoading loaderAnimation="border">Border</Button>
  <Button isLoading loaderAnimation="grow">Grow</Button>
</Preview>

```js
<Button isLoading loaderAnimation='border'>Border</Button>
<Button isLoading loaderAnimation='grow'>Grow</Button>
```

</article>

<article id='isLoader'>
### Hide Default Loader

The `isLoader` prop allows to toggle the visibility of the default loader or the custom loader provided via `loader`

<Preview>
  <Button isLoading>Default</Button>
  <Button isLoading isLoader={false}>! isLoader</Button>
</Preview>

```js
<Button isLoading isLoader={false}>
  Button
</Button>
```

</article>

<article id='customLoader'>
### Custom Loader

The loading icon can be replaced with `loader` prop.

<Preview>
  <Button isLoading loader={<CubeIcon animation="grow" size="sm" />}>Loading</Button>
</Preview>

```js
import Button from "components/button/Button";
import { ReactComponent as Loader } from "your-assets/loader.svg";

const CustomLoaderExample = () => {
  return (
    <Button isLoading loader={<Loader />}>
      Loading
    </Button>
  );
};
```
